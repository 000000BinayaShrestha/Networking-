# CCNA Networking Lab Journey (60 Days)

This repository documents my structured 60-day hands-on networking journey focused on building strong fundamentals aligned with CCNA-level concepts.

The goal of this project is to move beyond theory and develop practical networking skills using Cisco Packet Tracer, supported by structured documentation and progressive learning.

---

## 📌 Objective

- Build strong foundations in computer networking
- Understand real-world network behavior using simulations
- Develop practical skills in switching, routing, and network services
- Prepare for CCNA-level knowledge and networking roles

---

## 🧠 Learning Approach

- Hands-on practice using Cisco Packet Tracer
- Step-by-step documentation of concepts learned
- Gradual progression from basics to advanced topics
- Focus on understanding packet flow and troubleshooting

---

## 📅 Progress Overview

### Phase 1: Foundations & Switching (Days 1–14)

- OSI Model, ARP, MAC Addressing
- Binary, Hexadecimal, IP Addressing
- Subnetting (basic + requirement-based)
- Switching fundamentals
- VLANs and VLAN Trunking Protocol (VTP)
- STP (Spanning Tree Protocol)
- Rapid PVST+
- PortFast and BPDU Guard
- EtherChannel

### Phase 2: Routing (Current Focus)

- Static Routing
- Default Routing
- OSPF Fundamentals
- Inter-VLAN Routing

### Phase 3: Network Services

- DHCP
- DNS
- NAT / PAT
- ACLs
- NTP and Syslog basics

### Phase 4: Advanced Topics

- IPv6 Fundamentals
- STP Deep Dive
- EtherChannel optimization
- WAN concepts
- Port Security

### Phase 5: Capstone

- End-to-end network design
- Troubleshooting scenarios
- Full topology implementation

---

## 🛠 Tools Used

- Cisco Packet Tracer
- Networking simulation environments
- Documentation (Word / Markdown)
- Basic CLI configurations (Cisco IOS)
